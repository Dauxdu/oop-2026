# Практика 7. Полиморфмизм подтипов (часть 1)

Создайте класс `ICounter` (интерфейс) и три класса-наследника: `BoundedCounter`, `CyclicCounter`, `StepCounter`.

1. Интерфейс `ICounter`.
   Методы:
   - `increment`- увеличивает значение счётчика.
   - `reset` - сбрасывает значение счётчика в начальное (0).
   - `getValue` - возвращает текущее значение счётчика.
   - `isLimitReached` - возвращает true, если достигнут лимит (или следующий вызов `increment` приведёт к сбросу/остановке).
   - `getType` - возвращает строку с описанием типа счётчика и его параметров.

2. Класс `BoundedCounter` (счётчик с остановкой).
   - Параметры конструктора: максимальное значение.

   - Поведение: Значение увеличивается от 0 до максимального значения.
     После достижения лимита `increment` не изменяет значение (останавливается).

3. Класс `CyclicCounter` (циклический счётчик)
   - Параметры конструктора: максимальное значение.

   - Поведение: Значение увеличивается от 0 до максимального значения, после чего сбрасывается
     в 0 и продолжает рост.

4. Класс `StepCounter` (счётчик с шагом)
   - Параметры конструктора: максимальное значение, шаг.

   - Поведение: Аналог `BoundedCounter`, но с возможностью задать размер шага.

   - Примечание: Предусмотреть поведение в ситуации, когда шаг не кратен максимальному значению.

5. Продемонстрируйте работу классов.
